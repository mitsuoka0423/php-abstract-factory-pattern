# php-abstract-factory-pattern

# How to run

```sh
$ php Driver.php [list|table]
```

## Examples

```sh
$ php Driver.php list

- yahoo
  - Yahoo!Japan: https://yahoo.co.jp/
  - Yahoo!: https://yahoo.com/
```

```sh
$ php Driver.php table

| yahoo                                                 |
| caption              | url                            |
| Yahoo!Japan          | https://yahoo.co.jp/           |
| Yahoo!               | https://yahoo.com/             |
```
